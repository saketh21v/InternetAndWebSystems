Why, in a race track, are the athletes given different starting positions?

Asked by Sujit Semakurthi

The starting position is different for different type of race. For 100m all start from the same position and finish at the same position(here position is lateral sense with respect to all runners i.e. side by side),for 200m they are different and so on. The reason being the total distance the runner runs, this distance has to be equal for every runner. In 100m they run on a straight so the starting position and finishing positions are same for everyone. But in case of 200m or longer distances the track is part straight and part oval.
